## 📞 Quick Help

**Dashboard won't load?**
- Hard refresh: Cmd+Shift+R
- Clear cache or incognito window

**NFT not showing?**
- Check localStorage: DevTools → Application
- Verify tx hash on Sepolia

**Metrics not calculating?**
- Need ≥5 trades
- Check trade prices set
- Verify win_rate > 0

**Build failing?**
- `npm run build 2>&1 | tail -50`
- Fix TypeScript errors
- Check imports

**Questions?**
- Join Discord: discord.gg/lablabai
- Post in #dev-help
- Tag @lablab_ai
